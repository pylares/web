¿Qué puedo hacer con Pylares?

Este es un pequeño ejemplo de lo que se puede hacer usando Pylares:

            
# -*- coding: utf-8 -*-
from __future__ import unicode_literals
from pylares import *

opciones = ['Un avión', 'Un barco', 'Una bicicleta']
ventana = Ventana(
  Fila(
    Columna(
      Etiqueta("pregunta", "¿Qué es esto?"),
      Imagen("imagen", "bicicleta.png"),
    ),
    Columna(
      BotonesDeOpcion('eleccion', opciones, "Un barco"),
      Boton("jugar", "Elegir"),
      Entrada("resultado", "", "Resultado")
    )
  )
)

def ClicEn_jugar():
    if ventana["eleccion"].valor == 'Una bicicleta':
        ventana["resultado"].valor = "¡Correcto!"
    else:
        ventana["resultado"].valor = "No, intenta de nuevo..."

ventana.Dibujar()

Pylares es un conjunto de herramientas para hacer que la programación sea mucho más sencilla y divertida.

Con Pylares puedes crear programas con interfaces gráficas usando el lenguaje Python.

Pylares simplifica algunas operaciones del lenguaje Python para que sea más accesible a estudiantes adolescentes.

Muchas instrucciones de Python que están en inglés, fueron traducidas y simplificadas reduciendo de esta manera la necesidad de recordar que significan cada una de esas instrucciones.

Usando únicamente Python:

                    
# -*- coding: utf-8 -*-
from __future__ import unicode_literals
import random
import os

lista = ["rojo", "verde", "azul", "blanco"]

valor_aleatorio = random.randrange(0, len(lista))
color_secreto = lista[valor_aleatorio]

pudo_acertar = False

for i in range(10, 1, -1):
    color = raw_input("Elige un color: ")
    if color.lower() == color_secreto:
        print "Excelente, adivinaste"
        pudo_acertar = True
        break
    else:
        print "Ese no es el color que pensé"
        print "Te quedan", i, "intentos"
        print "Presione enter para continuar..."
        raw_input()
        if os.name == 'posix':
            os.system("clear")
        else:
            os.system("cls")

if pudo_acertar == False:
    print "Perdiste :("
    print "El color que pensé era el:", color_secreto

Usando Python con Pylares:

                
# -*- coding: utf-8 -*-
from __future__ import unicode_literals
from pylares import *

lista = ["Rojo", "Verde", "Azul", "Blanco"]

color_secreto = ElegirUnoDe(lista)
pudo_acertar = No

for i in DesdeHasta(10, 1):
    color = Leer("Elige un color: ")
    if Minuscula(color) == color_secreto:
        Mostrar("Excelente, adivinaste")
        pudo_acertar = Si
        break
    else:
        Mostrar("Ese no es el color que pensé")
        MostrarVarios("Te quedan", i, "intentos")
        Pausa()
        LimpiarPantalla()

if pudo_acertar == No:
    Mostrar("Perdiste :(")
    MostrarVarios("El color que pensé era el:", color_secreto)
